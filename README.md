## SendMe 


### Key Features:
    -> Login Screen.
    -> Registration Screen.
    -> Menu for HomePage, Settings and Logout.
    -> Realtime Authentication System.
    -> Chating users systems.
    -> Real time messege send and recieved.
    -> Password validation process.
